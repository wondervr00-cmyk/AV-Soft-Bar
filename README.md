
# AV Soft — Bares

Sistema SaaS multi-tenant de gestão de bares, desenvolvido pela AV Soft. Permite a bares e estabelecimentos gerir consumo de clientes, inventário e faturação de forma simples e digital.

## Funcionalidades

- 🎫 **Cartão de Consumo Digital** — sistema baseado em QR Code / código de barras para registo de consumo por cliente
- 🔒 **Anti-fraude** — validação de consumo em tempo real via Firestore
- 📦 **Gestão de Inventário** — controlo de produtos com fotos
- 🎨 **Personalização de Marca** — configurações de branding próprias para cada bar (logotipo, cores, nome)
- ☁️ **Multi-tenant** — vários bares podem usar a mesma plataforma, cada um com os seus dados isolados

## Tecnologia

- Frontend: HTML, CSS e JavaScript (ficheiro único)
- Backend: Firebase (Firestore + Storage)
- Hospedagem: GitHub Pages
